# 데이터 소스 레퍼런스

백테스팅 및 시세 분석을 위한 외부 데이터 소스 문서.

## 문서 목록

| 문서 | 설명 |
|---|---|
| [LS증권 Open API](ls-openapi.md) | 분봉 OHLCV 수집 (REST API) — **주력 데이터 소스** |
| [네이버 금융 API](naver-finance.md) | 일봉 OHLCV + 지수 데이터 (비공식 API) |
| [데이터 소스 비교](comparison.md) | 한국 주식 과거 분봉 데이터 소스 전체 비교 |

## 데이터 수집 전략

```
[LS증권 Open API]                  [네이버 금융]              [KIS API]
  │                                  │                        │
  ├─ 분봉 OHLCV (1년치)              ├─ 일봉 OHLCV (수 년치)    ├─ 당일 분봉 OHLCV
  ├─ 코스피/코스닥 지수 분봉           ├─ 코스피/코스닥 일봉       ├─ 실시간 체결가
  └─ 종목 마스터                     └─ 즉시 대량 수집 가능      └─ 매일 자동 수집
                      │
                ┌─────┴──────┐
                │ PostgreSQL  │
                │ (로컬 5433) │
                └─────────────┘
```
