## 프로젝트 채택 결과

### 주력: LS증권 Open API

- **선정 이유**: REST API로 분봉 OHLCV 완전 제공, 무료, Rust 직접 연동 가능
- **한계**: 1분봉 약 1년 범위
- **보완**: 매일 KIS API로 당일 분봉 자동 누적

### 보조: 네이버 금융

- **선정 이유**: 일봉 수 년치 즉시 수집, 코스피/코스닥 지수 지원, 무인증
- **한계**: 분봉 OHLC 미제공, 비공식 API

### 장기 고려: 대신증권 크레온

- **이유**: 5분봉 5년치로 가장 긴 분봉 이력
- **한계**: Windows COM 전용 → Python 수집 파이프라인 별도 구축 필요
- **시점**: LS증권 1년 데이터로 전략 검증 완료 후, 장기 백테스팅 필요 시 도입

## 데이터 수집 우선순위

```
1순위: LS증권 t8412 → 분봉 OHLCV 1년치 (PostgreSQL)
2순위: 네이버 금융 → 일봉 OHLCV 수 년치 (PostgreSQL)  
3순위: KIS API 당일 분봉 → 매일 자동 누적 (PostgreSQL)
4순위: (필요 시) 크레온 → 5분봉 5년치 보강
```
