# 기술지표 카탈로그

KIS Strategy Builder(`core/indicators.py`)에 구현된 96개 기술지표의 전체 목록이다. 우리 Rust 프로젝트에서 지표 엔진을 구현할 때 이 카탈로그를 참조하여 우선순위를 정한다.

> **출처**: https://github.com/koreainvestment/open-trading-api/blob/main/strategy_builder/core/indicators.py

## 구현 우선순위

- **P0 (핵심)**: 기본 매매 전략에 필수. 1차 개발 대상.
- **P1 (확장)**: 고급 전략에 사용. 2차 개발 대상.
- **P2 (전문)**: 특수 전략용. 필요 시 추가.

| 우선순위 | 개수 | 설명 |

### P0 지표 전체 목록 (1차 구현 대상)

```
기본: MA, EMA, 표준편차, 수익률, 이격도, 변동성, 등락률, N일최고/최저, 최근종가
RSI: RSI
MACD: MACD, 시그널, 히스토그램
볼린저: 중심/상단/하단/밴드폭/%B
스토캐스틱: %K, %D
거래량: OBV, VWAP, 거래량MA
기타: ATR, ROC
```
